# WarpGuardian Centinela

Mantiene **Cloudflare WARP (1.1.1.1) siempre conectado** en Windows: vigila cada 5 segundos, se adelanta a los cortes, se repara solo y te dice quien tuvo la culpa de cada caida.

## Descargar

**[Descargar el instalador para Windows](https://github.com/robertoaro8204-source/warpguardian/releases/latest/download/WarpGuardianSetup.exe)**

O visita la pagina: **https://robertoaro8204-source.github.io/warpguardian**

> Si Windows muestra `Windows protegio su PC`, no es un virus: es el aviso normal para apps nuevas sin firma de pago. Pulsa **Mas informacion -> Ejecutar de todas formas**.

## Que hace

- Vigilancia 24/7 del tunel WARP con reconexion automatica
- Anti-microcortes (desactiva el ahorro de energia de la tarjeta de red)
- Blindaje: Windows resucita el servicio en 2 segundos si algo lo mata
- Prediccion de cortes (indice de salud 0-100)
- Caja negra: identifica la causa de cada corte
- Panel movil en tu WiFi (solo lectura)

100% local. No recopila datos, sin anuncios, sin cuentas.
